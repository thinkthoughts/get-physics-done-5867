# Papers

This directory tracks physics papers explored through the
**get-physics-done-5867** workflow.

The goal is simple:

extract equations → run checks → record notes.

## Research Loop

observe → verify → revise

## Directory Structure

Each paper gets its own folder:

papers/
  arxiv-id/
    summary.md
    key-equations.md
    verification.py
    diagrams/

Example:

papers/
  2603.01575/
    summary.md
    key-equations.md
    verification.py
    diagrams/

## File Roles

summary.md  
Short overview of the paper.

key-equations.md  
Important equations extracted from the paper.

verification.py  
Numerical or symbolic checks.

diagrams/  
Figures or visual explanations.
